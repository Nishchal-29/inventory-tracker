# Inventory Tracker

Developed an inventory management RESTful API using FastAPI with real-time dashboard that displays live inventory updates, quantity changes, and item deletions.

Repo structure:
```text
inventory-tracker

├── models.py
├── notify.py
├── requirements.txt
├── setup_database.sql
└── static
    ├── index.html
    └── index.js
```
